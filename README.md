Job Portal Web Application — Portail d'Emploi en Ligne
By Oueslati Ouissem, Sarra Ben Bettaieb & Maram Raboudi
A full-stack web-based job portal built with Spring Boot and MySQL, designed to connect job seekers with recruiters through an intuitive and responsive interface.
The platform supports three user roles: job seekers can search for jobs, apply to listings, upload and manage their CV, and track their application history; recruiters can post and manage job offers and review candidate profiles; and administrators can oversee user accounts, validate listings, remove inappropriate content, and access usage statistics.
Tech stack: Java Spring Boot (MVC backend), Spring Security with BCrypt authentication, Thymeleaf, HTML/CSS/JavaScript, Bootstrap, and MySQL.
